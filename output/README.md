# Output

This folder contains the prediction results generated by the Linear Regression model.

The output file is created after training the model on historical customer data and predicting the purchase likelihood for new customers.

---

# File

### Car_Prediction.xlsx

This file contains the prediction results for new customers.

Each record represents a customer along with the model's estimated likelihood of purchasing a hybrid vehicle.

---

# Output Columns

The output includes the following information:

- Age
- Education
- Income (Lacs)
- Vehicle Ownership
- Predicted Purchase Percentage

The **Predicted Purchase Percentage** is generated using the trained Linear Regression model.

---

# How the Output is Generated

```text
Training Data
      │
      ▼
Train Linear Regression Model
      │
      ▼
Load New Customer Dataset
      │
      ▼
Predict Purchase Likelihood
      │
      ▼
Export Results
      │
      ▼
Car_Prediction.xlsx
```

The prediction results can help businesses:
