# Excel Analysis

This folder contains the Microsoft Excel implementation of the **Hybrid Vehicle Purchase Prediction** project.

The purpose of this workbook is to understand the mathematical concepts behind **Linear Regression** before implementing the same model using Python and Scikit-learn.

---

## File

### Linear Regression.xlsx

This workbook demonstrates how Linear Regression can be performed using **Microsoft Excel Analysis ToolPak**.

The workbook is divided into multiple sections:

### 1. Customer Dataset

Contains the historical customer information used for regression analysis.

Features include:

- Age
- Education
- Income (Lacs)
- Vehicle Ownership
- Purchase Hybrid Vehicle (Target Variable)

---

### 2. Regression Analysis

Performed using **Excel Data Analysis ToolPak**.

The output includes:

- Regression Statistics
- Multiple R
- R Square
- Adjusted R Square
- Standard Error
- ANOVA Table
- Coefficient Table
- Intercept

These statistics help evaluate the relationship between customer characteristics and hybrid vehicle purchases.

---

### 3. Manual Prediction

Using the regression equation:

**Y = a + bX**

where:

- **a** = Intercept
- **b** = Regression Coefficients
- **X** = Customer Features

Predictions were calculated manually to understand how each feature contributes to the final prediction.

## Purpose

This Excel workbook demonstrates that Linear Regression can be performed without programming by using Microsoft Excel's built-in analytical tools.

Understanding the mathematical concepts in Excel made it easier to implement the same predictive model later using Python and Scikit-learn.
